## Image Optimization Guidelines

After generating images with Nano Banana:

1. **Resize** to appropriate dimensions:
   - Hero images: 1920x1080px
   - Section images: 1200x800px
   - Background textures: 1920x1080px

2. **Optimize** file size:
   - Use JPEG format with 80-85% quality
   - Target file size: 150-300KB for web performance

3. **Color Correction**:
   - Ensure warm tones align with brand palette
   - Adjust to complement gold ochre (#CBA85C) and warm stone (#DBD5CA)

4. **File Naming**:
   - Use lowercase with hyphens
   - Be descriptive and consistent
   - Example: `hero-hospitals.jpg`, `case-study-bmc.jpg`
